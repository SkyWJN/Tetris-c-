# Tetris-c-
纯c++的俄罗斯方块小游戏
利用easyx图形库设计俄罗斯方块小游戏  
小键盘方向键控制，← →：控制方块左右移动；↑：旋转方块；↓：快速下降方块。  
消除行会得分，使用ifstream头文件将最高分记录到txt文件内。有背景音乐和消除音效。
